2. **Create src/grade_results.py** with these components:

- `GradeResult` frozen dataclass:
  - `passed: bool`, `fail_reason: str | None`, `stdout: str`, `stderr: str`, `execution_time_ms: float`, `extraction_method: str | None`

- `extract_code(response: str) -> str`: Strip markdown fences using `re.findall(r'```(?:python)?\s*\n(.*?)```', response, re.DOTALL)`, return `blocks[-1].strip()` if found, else `response.strip()`.

- `_set_limits() -> None`: preexec_fn that calls `os.setsid()` (for process group kill), then `resource.setrlimit(resource.RLIMIT_AS, (512 * 1024 * 1024, 512 * 1024 * 1024))`, `resource.setrlimit(resource.RLIMIT_NPROC, (50, 50))` (50 not 10 per research open question), `resource.setrlimit(resource.RLIMIT_CPU, (15, 15))`.

- `_run_sandbox(code: str, timeout: float = 10.0) -> subprocess.CompletedProcess`: Create tempdir, write code to `solution.py`, run `subprocess.run([sys.executable, script_path], capture_output=True, text=True, timeout=timeout, cwd=tmpdir, preexec_fn=_set_limits)`. On `TimeoutExpired`, kill process group via `os.killpg(os.getpgid(proc.pid), signal.SIGKILL)`. Use `os.setsid()` inside `_set_limits` instead of `start_new_session=True` to avoid the Python compatibility issue noted in research.

- `_build_humaneval_harness(llm_code: str, test_code: str, entry_point: str) -> str`: Return `f"{llm_code}\n\n{test_code}\n\ncheck({entry_point})\n"`.

- `_build_mbpp_harness(llm_code: str, test_code: str) -> str`: Return `f"{llm_code}\n\n{test_code}\n"`.

- `_extract_entry_point(prompt_text: str) -> str | None`: Use `re.search(r'def (\w+)\(', prompt_text)` to get the function name from the PROMPT (not the LLM output).

- `grade_code(raw_output: str, prompt_record: dict) -> GradeResult`:
  - If raw_output is empty/whitespace, return fail with fail_reason="no_output"
  - Extract code via `extract_code(raw_output)`
  - Optionally ast.parse() pre-check for syntax errors (return fail_reason="syntax_error")
  - Build harness based on `prompt_record["benchmark_source"]`: "humaneval" uses `_build_humaneval_harness` with entry_point from `_extract_entry_point(prompt_record["prompt_text"])`, "mbpp" uses `_build_mbpp_harness`
  - Run `_run_sandbox(harness_code)`
  - Map results: returncode 0 = passed=True, TimeoutExpired = fail_reason="timeout", non-zero with "MemoryError" or signal 9 in stderr = "memory_exceeded", "SyntaxError" in stderr = "syntax_error", "ImportError"/"ModuleNotFoundError" in stderr = "import_error", other non-zero = "crash"
  - Record wall-clock execution_time_ms via time.monotonic()

1. **Add to src/grade_results.py** -- GSM8K grading components:

- `ExtractionResult` dataclass (not frozen -- intermediate value):
  - `value: float`, `method: str`, `raw_match: str`

- `_normalize_number(raw: str) -> float`: Normalize raw number string to float. Handle parenthetical negatives `re.match(r'^\((\d+\.?\d*)\)$', s)` -> negate. Handle fractions `re.match(r'^(-?\d+)/(\d+)$', s)` -> divide. Handle percentages (strip trailing %). Strip currency prefix ($). Strip unit suffixes (dollars, meters, hours, minutes, seconds, kg, lbs, miles, feet, cm, mm, m) via `re.sub(r'\s*(dollars?|meters?|hours?|minutes?|seconds?|kg|lbs?|miles?|feet|cm|mm|m)\s*$', '', s, flags=re.IGNORECASE)`. Strip commas. Return `float(s.strip())`.

- `_extract_number(text: str) -> ExtractionResult | None`:
  1. If text is empty/whitespace, return None
  2. Check for LaTeX `\boxed{...}` -- `re.findall(r'\\boxed\{([^}]+)\}', text)`, take last match, normalize, return with method="latex_boxed"
  3. Find ALL number-like patterns using these regexes in order (compile once at module level):
     - `r'[-]?\d{1,3}(?:,\d{3})+(?:\.\d+)?'` -- comma_separated (1,234.56)
     - `r'[-]?\d+/\d+' ` -- fraction (3/4)
     - `r'[-]?\d+\.?\d*\s*%'` -- percentage (50%)
     - `r'\((\d+\.?\d*)\)'` -- paren_negative (42)
     - `r'[-]?\d+\.\d+'` -- decimal (3.14)
     - `r'[-]?\d+'` -- integer (42)
  4. Collect all matches with their position in text. Take the LAST match by position.
  5. Normalize via `_normalize_number`, return ExtractionResult with appropriate method string.

- `grade_math(raw_output: str, prompt_record: dict) -> GradeResult`:
  - If raw_output is empty/whitespace, return fail with fail_reason="no_output"
  - Extract number via `_extract_number(raw_output)`
  - If None, return fail with fail_reason="extraction_failed"
  - Parse canonical answer: `float(prompt_record["canonical_answer"])`
  - Compare with epsilon: `abs(extracted.value - canonical) < 1e-6`
  - If match: passed=True, extraction_method=extracted.method
  - If no match: passed=False, fail_reason="wrong_answer", extraction_method=extracted.method
  - Record execution_time_ms (trivial but consistent with code grading)
