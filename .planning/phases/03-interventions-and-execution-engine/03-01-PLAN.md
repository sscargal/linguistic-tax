   ```python
   SELF_CORRECT_PREFIX: str = (
       "Note: my prompt below may contain spelling or grammar errors. "

   )

   _COMPRESS_INSTRUCTION = (

   )
   ```

4. `def _get_preproc_model(main_model: str) -> str:` looks up PREPROC_MODEL_MAP, raises ValueError if not found.

5. `def sanitize(text: str, main_model: str, call_fn: Callable) -> tuple[str, dict]:`
   - Looks up preproc_model via _get_preproc_model(main_model)
   - Calls call_fn(model=preproc_model, system=_SANITIZE_SYSTEM, user_message=_SANITIZE_INSTRUCTION + text, max_tokens=max(512, len(text) * 2), temperature=0.0)
   - Builds metadata dict with preproc_model, preproc_input_tokens, preproc_output_tokens, preproc_ttft_ms, preproc_ttlt_ms from response
   - Fallback: if result.strip() is empty or len(result.strip()) > len(text) * 1.5, log WARNING, set metadata["preproc_failed"] = True, return (text, metadata)
   - Otherwise return (result.strip(), metadata)

6. `def sanitize_and_compress(text: str, main_model: str, call_fn: Callable) -> tuple[str, dict]:` — same pattern with _COMPRESS_SYSTEM and _COMPRESS_INSTRUCTION.
