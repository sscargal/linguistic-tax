  <action>
Create src/compute_derived.py with these exact functions:

1. **compute_cr(pass_fail_results: list[int]) -> float:**
   - Uses itertools.combinations to enumerate all C(K,2) pairs
   - Counts agreeing pairs (both 1 or both 0)
   - Returns agreeing_pairs / total_pairs
   - If K < 2: return 1.0 (degenerate case)

2. **classify_quadrant(cr: float, majority_pass: bool, cr_threshold: float = 0.8) -> str:**
   - stable = cr >= cr_threshold
   - Returns: stable+pass="robust", stable+fail="confidently_wrong", unstable+pass="lucky", unstable+fail="broken"

3. **build_condition_string(noise_type: str, intervention: str) -> str:**
   - Returns f"{noise_type}_{intervention}" (e.g., "clean_raw", "type_a_10pct_pre_proc_sanitize")

4. **load_experiment_data(db_path: str) -> pd.DataFrame:**
   - sqlite3.connect, pd.read_sql_query
   - Query: `SELECT * FROM experiment_runs WHERE status = 'completed' AND pass_fail IS NOT NULL`
   - Close connection and return DataFrame

5. **compute_derived_metrics(db_path: str, cr_threshold: float = 0.8) -> dict:**
   - Load data via load_experiment_data
   - Group by (prompt_id, noise_type, intervention, model)
   - For each group:
     - Extract pass_fail values as list[int]
     - If fewer than 5 runs: log WARNING with prompt_id, compute CR with actual K, set flag
     - Compute CR via compute_cr
     - Compute majority_pass = sum(pass_fail) >= len(pass_fail) / 2 (i.e., 3/5+)
     - Compute pass_count = sum(pass_fail)
     - Classify quadrant via classify_quadrant(cr, majority_pass, cr_threshold)
     - Compute mean_ttft_ms, mean_ttlt_ms, mean_total_latency_ms (ttft+ttlt), std_latency_ms
     - Compute mean_total_cost_usd from total_cost_usd
     - Compute token_savings = mean(prompt_tokens - optimized_tokens) across reps (0 if optimized_tokens is NULL)
     - Compute net_token_cost = mean(prompt_tokens) - token_savings (net tokens after optimization)
     - Build condition string via build_condition_string
     - INSERT OR REPLACE into derived_metrics table
   - Return summary dict with counts per quadrant, total prompts processed, incomplete groups

6. **compute_quadrant_migration(db_path: str, model: str, from_condition: str, to_condition: str) -> dict:**
   - Query derived_metrics for both conditions
   - Match prompts by prompt_id
   - Build 4x4 transition matrix: {"robust": {"robust": N, "lucky": N, ...}, ...}
   - Return dict with from_condition, to_condition, model, n_prompts, transition_matrix

7. **compute_cost_rollups(db_path: str) -> list[dict]:**
   - Query experiment_runs grouped by (model, noise_type, intervention)
   - For each group compute: n_runs, mean_total_cost_usd, sum_total_cost_usd, mean_preproc_cost_usd, sum_preproc_cost_usd, mean_main_cost_usd (input+output), mean_token_savings
   - Return list of rollup dicts

8. **main() -> None:**
   - argparse with: --db (default "results/results.db"), --cr-threshold (default 0.8, type=float), --output-dir (default "results")
   - Call compute_derived_metrics (writes to DB, prints summary via tabulate)
   - Call compute_cost_rollups (writes JSON to {output_dir}/cost_rollups.json, CSV to {output_dir}/csv/cost_rollups.csv)
   - Call compute_quadrant_migration for all clean->noisy transitions (clean_raw -> type_a_5pct_raw, clean_raw -> type_a_10pct_raw, clean_raw -> type_a_20pct_raw) for each model
   - Write migration results to {output_dir}/quadrant_migration.json
   - Write quadrant distribution CSV to {output_dir}/csv/quadrant_distribution.csv
   - Print terminal summary tables using tabulate(format="grid")
   - if __name__ == "__main__": main()

All functions use logging module (no print). All functions have type hints and docstrings.
