**Module-level setup:**
- `matplotlib.use("Agg")` BEFORE importing pyplot (prevents display backend errors)
- `import matplotlib.pyplot as plt`, `import seaborn as sns`, `import pandas as pd`, `import numpy as np`

**_configure_style() function** (called at module level):
- `sns.set_theme(style="whitegrid", palette="colorblind")`
- rcParams: axes.labelsize=12, axes.titlesize=14, xtick.labelsize=10, ytick.labelsize=10, legend.fontsize=10, figure.dpi=300, savefig.dpi=300, savefig.bbox="tight", savefig.pad_inches=0.1, pdf.fonttype=42, ps.fonttype=42
- Call `_configure_style()` immediately after definition

**_save_figure(fig, output_dir, name, fmt="both") -> list[str]:**
- os.makedirs(output_dir, exist_ok=True)
- Save PDF if fmt in ("pdf", "both"), PNG if fmt in ("png", "both")
- plt.close(fig) after saving
- Return list of saved file paths

**generate_accuracy_curves(db_path, output_dir="figures", fmt="both", analysis_dir="results/analysis") -> list[str]:**
- Load experiment data via pd.read_sql_query from experiment_runs table (filter status='completed' AND pass_fail IS NOT NULL)
- Filter to Type A noise levels: clean, type_a_5pct, type_a_10pct, type_a_20pct
- Compute per-condition accuracy: groupby(model, intervention, noise_type)["pass_fail"].mean()
- Load bootstrap CIs from analysis_dir/csv/bootstrap_cis.csv if available
- Create fig with plt.subplots(1, 2, figsize=(7, 3.5), sharey=True) -- one panel per model
- Plot lines per intervention with markers; add fill_between for CI bands if available
- Set x-axis labels as ["Clean", "5%", "10%", "20%"], y-axis as "Accuracy"
- Legend on rightmost panel
- fig.tight_layout()
- Return _save_figure(fig, output_dir, "robustness_curve", fmt)

**generate_quadrant_plot(db_path, output_dir="figures", fmt="both") -> list[str]:**
- Load derived_metrics via pd.read_sql_query from derived_metrics table
- Create fig with plt.subplots(1, 2, figsize=(7, 3.5)) -- one panel per model
- Scatter: x=majority_pass (jittered slightly for visibility), y=consistency_rate, colored by quadrant
- Draw quadrant boundaries: axhline at y=0.8 (gray dashed), axvline at x=0.5 (gray dashed)
- Annotate each quadrant with count (n=X) in the quadrant's center area
- Labels: x="Majority Pass Rate", y="Consistency Rate (CR)"
- Return _save_figure(fig, output_dir, "quadrant_migration", fmt)

**generate_cost_heatmap(db_path, output_dir="figures", fmt="both", analysis_dir="results/analysis") -> list[str]:**
- Load cost_rollups.json from analysis_dir, convert to DataFrame
- Pivot: index=intervention, columns=model, values=mean_token_savings, aggfunc="mean"
- Create fig with plt.subplots(figsize=(3.5, 3.5)) -- single-column
- sns.heatmap(pivot, annot=True, fmt=".0f", center=0, cmap=sns.diverging_palette(10, 130, as_cmap=True), linewidths=0.5)
- Title: "Net Token Savings by Condition"
- Return _save_figure(fig, output_dir, "cost_model", fmt)

**generate_kendall_plot(db_path, output_dir="figures", fmt="both", analysis_dir="results/analysis") -> list[str]:**
- Load kendall_results.json from analysis_dir, convert to DataFrame
- Create fig with plt.subplots(figsize=(3.5, 3.5)) -- single-column
- sns.barplot: x=noisy_condition, y=tau, hue=model
- If ci_lower and ci_upper columns exist, add error bars manually via ax.errorbar
- axhline at y=1.0 (gray dashed) as "Perfect agreement" reference
- y-axis: "Kendall's tau", x-axis: "Condition", ylim=(0, 1.05)
- Rotate x-tick labels 45 degrees
- Return _save_figure(fig, output_dir, "rank_stability", fmt)

**CLI (main function with argparse):**
- Match Phase 5 pattern: argparse with subcommands (accuracy, quadrant, cost, kendall, all)
- Common flags: --db (default "results/results.db"), --output-dir (default "figures"), --format choices=["pdf", "png", "both"] default="both", --analysis-dir (default "results/analysis")
- "all" subcommand calls all 4 generate functions
- Each subcommand calls its respective function
- if __name__ == "__main__": main()
- Configure logging at INFO level in main()
