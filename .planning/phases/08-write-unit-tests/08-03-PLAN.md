Create `scripts/qa_script.sh` with `#!/usr/bin/env bash` and `set -uo pipefail` (NOT -e, because we handle failures per-check).

**Global variables and argument parsing:**
```bash
PASS_COUNT=0; FAIL_COUNT=0; WARN_COUNT=0; INFO_COUNT=0; CHECK_NUM=0
LIVE=false; LOG_FILE=""; SECTION="all"
START_TIME=$(date +%s)
SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PROJECT_ROOT="$(cd "$SCRIPT_DIR/.." && pwd)"
```

Parse args: `--live` sets LIVE=true, `--section <name>` sets SECTION (valid: env, pytest, cli, data, config, api, all), `--log` sets LOG_FILE to `qa_results_$(date +%Y%m%d_%H%M%S).log`. Show usage on `--help` or `-h`.

**Helper functions:**

`run_check()` -- takes description string and command. Runs command, measures elapsed time in ms, prints formatted row: `| NNN | STATUS | TIMEms | description |`. Increments appropriate counter. STATUS is PASS (green) or FAIL (red). If LOG_FILE is set, also append to log (strip ANSI colors).

`run_warn_check()` -- like run_check but non-fatal: prints WARN (yellow) instead of FAIL on failure. Increments WARN_COUNT.

`run_info()` -- prints INFO (blue) line with description, no pass/fail. Increments INFO_COUNT.

`print_header()` -- prints section header with divider line.

`print_summary()` -- prints final summary table:
```
============================================
QA SUMMARY
============================================
PASS: NN | FAIL: NN | WARN: NN | INFO: NN
Total checks: NN
Duration: NNs
--------------------------------------------
VERDICT: PASS (or FAIL)
============================================
```

**Main flow:**
```bash

print_header "Linguistic Tax QA Script"

if [[ "$SECTION" == "all" || "$SECTION" == "env" ]]; then check_env; fi
if [[ "$SECTION" == "all" || "$SECTION" == "pytest" ]]; then check_pytest; fi
if [[ "$SECTION" == "all" || "$SECTION" == "cli" ]]; then check_cli; fi
if [[ "$SECTION" == "all" || "$SECTION" == "data" ]]; then check_data; fi

if [[ "$SECTION" == "api" || ( "$SECTION" == "all" && "$LIVE" == true ) ]]; then check_api; fi

print_summary
exit $( [ "$FAIL_COUNT" -gt 0 ] && echo 1 || echo 0 )
```
