**Section 9: Glossary**
Alphabetical within two groups per RESEARCH.md recommendation:

Research Concepts:
- Consistency Rate (CR) -- pairwise agreement across 5 repetitions per condition. Values: 0.0 (no agreement) to 1.0 (all identical). Code: `src/compute_derived.py`
- GLMM -- Generalized Linear Mixed Model with prompt-level random effects on binary pass/fail. Code: `src/analyze_results.py`
- Kendall's tau -- rank-order stability metric comparing uniform vs targeted noise degradation. Code: `src/analyze_results.py`
- Linguistic Tax -- accuracy degradation caused by prompt noise (typos, ESL patterns, verbosity)
- McNemar's test -- paired test for prompt-level fragility/recoverability with BH correction. Code: `src/analyze_results.py`
- Quadrant Classification -- each prompt-condition classified as Robust (high accuracy + high CR), Confidently Wrong (low accuracy + high CR), Lucky (high accuracy + low CR), or Broken (low accuracy + low CR). Code: `src/compute_derived.py`
- Type A noise -- character-level typos (adjacent key swaps, insertions, deletions, transpositions) at 5/10/20% rates. Code values: `type_a_5pct`, `type_a_10pct`, `type_a_20pct`
- Type B noise -- ESL syntactic patterns from L1 transfer (article errors, verb tense, word order). Code values: `type_b_mandarin`, `type_b_spanish`, `type_b_japanese`, `type_b_mixed`

Technical Terms:
- Experiment matrix -- full factorial design of all prompt x noise x intervention x model combinations (~82K items). File: `data/experiment_matrix.json`
- Intervention -- strategy applied to a prompt before sending to the target model. 5 types: Raw, Self-Correct, Pre-Proc Sanitize, Pre-Proc Sanitize+Compress, Prompt Repetition
- Pre-processor model -- cheap/fast model (Haiku, Flash, GPT-4o-mini, Nemotron-nano) used to sanitize/compress prompts before sending to the target model
- Target model -- the LLM being evaluated (Sonnet, Gemini Pro, GPT-4o, Nemotron-super)
- Work item -- single row in the experiment matrix: one prompt + noise type + intervention + model + repetition number
