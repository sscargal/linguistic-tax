    Create `src/config_manager.py` with these exact exports:

    1. `CONFIG_FILENAME = "experiment_config.json"` -- the config file name (avoids collision with .planning/config.json)

    2. `find_config_path(start_dir: str = ".") -> Path | None` -- looks for CONFIG_FILENAME in start_dir, returns Path if exists, None otherwise

    3. `load_config(config_path: Path | None = None) -> ExperimentConfig` -- if config_path is None, calls find_config_path(). If no file found, returns ExperimentConfig() (all defaults). If file found, reads JSON, filters to valid ExperimentConfig field names using `dataclasses.fields()`, converts list values to tuples for fields whose type annotation contains "tuple", constructs and returns ExperimentConfig(**filtered). Unknown JSON keys are silently ignored.

    4. `save_config(config: dict, config_path: Path | None = None) -> Path` -- writes config dict to JSON with indent=2. If config_path is None, uses Path(".") / CONFIG_FILENAME. Converts tuple values to lists before serializing. Returns the path written to.

    5. `get_full_config_dict() -> dict` -- creates ExperimentConfig(), calls dataclasses.asdict(), converts tuple values to lists, returns the dict. This gives the full set of defaults for writing a complete config file.

    6. `validate_config(config_dict: dict) -> list[str]` -- validates config values, returns list of error message strings (empty = valid). Checks:
       - Model fields (claude_model, gemini_model, openai_model, openrouter_model, openrouter_preproc_model): if present, value must be a key in PRICE_TABLE
       - type_a_rates: if present, every rate must be 0 <= rate <= 1
       - type_a_weights: if present, every weight must be 0 <= weight <= 1
       - repetitions: if present, must be >= 1
       - temperature: if present, must be >= 0
       - prompts_path, matrix_path: if present and file does not exist, add error (these are data files that should exist)
       - results_db_path: if present, parent directory must exist or be creatable (don't error -- it gets created)

    Use `from src.config import ExperimentConfig, PRICE_TABLE` and `from dataclasses import fields, asdict`. Use `from pathlib import Path`. Use `import json` and `import logging`.

    defaults = ExperimentConfig()
