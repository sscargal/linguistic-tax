**Helper: `_load_raw_overrides(config_path=None) -> tuple[dict, Path]`**
- If config_path is None, call `find_config_path()`
- If path exists, read JSON and return `(dict, path)`
- If path is None or does not exist, return `({}, Path(".") / CONFIG_FILENAME)`
- This reads the SPARSE overrides only, NOT the merged config

**Helper: `_format_value(value) -> str`**
- If value is a tuple or list, format as comma-separated: `"0.05, 0.10, 0.20"`
- Otherwise return `str(value)`

**Helper: `_coerce_value(field_name: str, raw_value: str)`**
- Get default via `getattr(ExperimentConfig(), field_name)`
- If `isinstance(default_val, bool)`: return `raw_value.lower() in ("true", "1", "yes")`
- If `isinstance(default_val, int)`: return `int(raw_value)`
- If `isinstance(default_val, float)`: return `float(raw_value)`
- If `isinstance(default_val, tuple)`:
  - Split on commas: `parts = [p.strip() for p in raw_value.split(",")]`
  - If all elements of default are float: `return tuple(float(p) for p in parts)`
  - Else: `return tuple(parts)`
- Else: return `raw_value` (str)
- Wrap in try/except ValueError, print error and sys.exit(1)

**`handle_show_config(args) -> None`:**
- Load current config via `load_config()`
- Create defaults via `ExperimentConfig()`
- If `args.property` is set (single property query):
  - Validate property name exists in ExperimentConfig fields, exit 1 if not
  - Get value via `getattr(current, args.property)`
  - If `args.json`: print `json.dumps({args.property: value}, default=_json_default)`
  - Else: print `_format_value(value)`
  - Return early
- Build rows: for each `dc_fields(ExperimentConfig)`:
  - cur_val = getattr(current, f.name)
  - def_val = getattr(defaults, f.name)
  - modified = "*" prefix if cur_val != def_val, else ""
  - Row: [f"{modified}{f.name}", _format_value(cur_val), _format_value(def_val)]
  - If `args.verbose`: append FIELD_DESCRIPTIONS.get(f.name, "") as 4th column
- If `args.changed`: filter rows to those starting with "*"
- If `args.json`:
  - Build dict of all (or changed-only) properties
  - print `json.dumps(config_dict, indent=2, default=_json_default)`
- Else:
  - headers = ["Property", "Value", "Default"] (+ "Description" if verbose)
  - print `tabulate(rows, headers=headers, tablefmt="simple")`

**`_json_default(obj)`** helper: if tuple, return list; raise TypeError otherwise.

**`handle_set_config(args) -> None`:**
- Get pairs from `args.pairs` (list of strings)
- If `len(pairs) % 2 != 0`: print error, sys.exit(1)
- Load raw overrides via `_load_raw_overrides()`
- Get valid_names = `{f.name for f in dc_fields(ExperimentConfig)}`
- defaults = ExperimentConfig()
- For each key-value pair (step 2):
  - If key not in valid_names: print `f"Error: unknown property '{key}'"`, sys.exit(1)
  - Coerce value via `_coerce_value(key, value_str)`
  - Store in changes dict
- Merge: `merged = {**raw, **changes}`
- Validate: `errors = validate_config(merged)`
- If errors: print each, sys.exit(1)
- Print change summary for each key: `f"{key}: {_format_value(old_val)} -> {_format_value(new_val)}"`
  - old_val = raw.get(key, getattr(defaults, key))
- Save: `save_config(merged, config_path)`

**`handle_reset_config(args) -> None`:**
- Load raw overrides via `_load_raw_overrides()`
- If `args.all`:
  - If config_path exists: delete file, print "Config reset to defaults"
  - Else: print "No config file found, already using defaults"
  - Return
- valid_names = `{f.name for f in dc_fields(ExperimentConfig)}`
- For each property in `args.properties`:
  - If property not in valid_names: print error, sys.exit(1)
  - If property in raw: `del raw[property]`, print `f"{property}: reset to default"`
  - Else: print `f"{property}: already at default"`
- Save: `save_config(raw, config_path)`

**`handle_validate(args) -> None`:**
- Load raw overrides via `_load_raw_overrides()`
- errors = `validate_config(raw)`
- If errors: print each prefixed with "Error: ", sys.exit(1)
- Else: print "Configuration is valid"

**`handle_diff(args) -> None`:**
- Load current via `load_config()`, defaults via `ExperimentConfig()`
- rows = []
- For each field in `dc_fields(ExperimentConfig)`:
  - cur_val = getattr(current, f.name)
  - def_val = getattr(defaults, f.name)
  - If cur_val != def_val: rows.append([f.name, _format_value(def_val), _format_value(cur_val)])
- If no rows: print "No changes from defaults"
- Else: print `tabulate(rows, headers=["Property", "Default", "Current"], tablefmt="simple")`

**`handle_list_models(args) -> None`:**
- Build rows from sorted PRICE_TABLE.items():
  - model name
  - cost string: "free" if both input and output are 0, else `f"${inp:.2f} / ${out:.2f}"`
- Print `tabulate(rows, headers=["Model", "Input / Output (per 1M tokens)"], tablefmt="simple")`

**`property_name_completer(prefix, parsed_args, **kwargs) -> list[str]`:**
- Return `[f.name for f in dc_fields(ExperimentConfig) if f.name.startswith(prefix)]`
