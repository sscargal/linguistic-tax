    **Step 2: Create `src/model_registry.py`**

    Implement these components:

    1. `ModelConfig` dataclass (mutable, NOT frozen):
       - Fields: `model_id: str`, `provider: str`, `role: str`, `preproc_model_id: str | None = None`, `input_price_per_1m: float | None = None`, `output_price_per_1m: float | None = None`, `rate_limit_delay: float | None = None`
       - Type hints on all fields, docstring on class

    2. `_load_default_models() -> list[ModelConfig]` function:
       - Load `data/default_models.json` using `Path(__file__).resolve().parent.parent / "data" / "default_models.json"` (relative to src/)
       - Parse each entry into a ModelConfig instance
       - Return list of ModelConfig

    3. `ModelRegistry` class:
       - `__init__(self, models: list[ModelConfig])`: Store models in `self._models: dict[str, ModelConfig]` keyed by model_id. Initialize `self._warned_unknown: set[str]` for once-per-model warnings.
       - `get_price(self, model_id: str) -> tuple[float, float]`: Return (input_per_1m, output_per_1m). For unknown models, return (0.0, 0.0). For known models with None pricing, return 0.0 for the None field. Use `price if price is not None else 0.0` -- NEVER use `price or 0.0`.
       - `get_preproc(self, model_id: str) -> str | None`: Return preproc_model_id for target, None if not found or not a target.
       - `get_delay(self, model_id: str) -> float`: Return rate_limit_delay. Default 0.5 for unknown models. Use `delay if delay is not None else 0.5`.
       - `target_models(self) -> list[str]`: Return model_ids where role == "target".
       - `compute_cost(self, model_id: str, input_tokens: int, output_tokens: int) -> float`: If model unknown, log warning ONCE via `_warned_unknown` set, return 0.0. Warning message: `"Unknown model '%s': using $0.00 pricing. Run 'propt list-models' to see configured models."`. If known, compute: `input_tokens * inp / 1_000_000 + output_tokens * out / 1_000_000`.
       - `reload(self, models: list[ModelConfig]) -> None`: Replace `self._models` dict, clear `self._warned_unknown`.
       - `check_provider(self, provider: str) -> dict[str, bool]`: Look up env var name from hardcoded map `{"anthropic": "ANTHROPIC_API_KEY", "google": "GOOGLE_API_KEY", "openai": "OPENAI_API_KEY", "openrouter": "OPENROUTER_API_KEY"}`. Return `{"key_name": env_var_name, "exists": bool(os.environ.get(env_var_name, ""))}`. For unknown providers, return `{"key_name": f"{provider.upper()}_API_KEY", "exists": False}`.

    4. Module-level singleton:
       - `def _build_registry() -> ModelRegistry`: Call `_load_default_models()`, return `ModelRegistry(models)`.
       - `registry: ModelRegistry = _build_registry()` at module level.
       - This initializes from defaults. config_manager.py will call `registry.reload()` after loading config (Plan 03).

    Use `import logging; logger = logging.getLogger(__name__)` for all logging.
    Use `import json, os; from pathlib import Path; from dataclasses import dataclass`.
