Create `src/model_discovery.py` with:

1. **DiscoveredModel dataclass:**
```python
@dataclass
class DiscoveredModel:
    model_id: str
    provider: str
    context_window: int | None
    input_price_per_1m: float | None
    output_price_per_1m: float | None
```

2. **DiscoveryResult dataclass:**
```python
@dataclass
class DiscoveryResult:
    models: dict[str, list[DiscoveredModel]]  # provider -> models
    errors: dict[str, str]  # provider -> error message
```

3. **_query_anthropic(timeout: float = 5.0) -> list[DiscoveredModel]:**
   - `import anthropic`
   - Create `anthropic.Anthropic(timeout=timeout)`
   - Call `client.models.list(limit=100)`, iterate `.data`
   - Handle pagination: while `page.has_more`, call `client.models.list(limit=100, after_id=page.last_id)`
   - Map: `model_info.id` -> model_id, `model_info.max_input_tokens` -> context_window, pricing = None/None

4. **_query_google(timeout: float = 5.0) -> list[DiscoveredModel]:**
   - `from google import genai`
   - Create `genai.Client(api_key=os.environ["GOOGLE_API_KEY"])`
   - Iterate `client.models.list()`
   - Strip "models/" prefix: `(model.name or "").removeprefix("models/")`
   - Skip empty model_id
   - Map: `model.input_token_limit` -> context_window, pricing = None/None

5. **_query_openai(timeout: float = 5.0) -> list[DiscoveredModel]:**
   - `import openai`
   - Create `openai.OpenAI(timeout=timeout)`
   - Call `client.models.list()`, iterate
   - Map: `model.id` -> model_id, context_window = None, pricing = None/None

6. **_query_openrouter(timeout: float = 5.0) -> list[DiscoveredModel]:**
   - `import requests`
   - `from src.config import OPENROUTER_BASE_URL`
   - Headers: `{"Authorization": f"Bearer {api_key}"}` if api_key set
   - `requests.get(f"{OPENROUTER_BASE_URL}/models", headers=headers, timeout=timeout)`
   - Parse `resp.json()["data"]`
   - Pricing conversion: `float(pricing["prompt"]) * 1_000_000` for per-1M
   - Free models: pricing string "0" -> `0.0`
   - Map: `item["id"]` -> model_id, `item.get("context_length")` -> context_window

7. **_PROVIDER_QUERY_MAP dict** mapping "anthropic" -> _query_anthropic, etc.

8. **discover_all_models(timeout: float = 5.0) -> DiscoveryResult:**
   - Import `_PROVIDER_KEY_MAP` from `src.model_registry`
   - For each provider in `_PROVIDER_QUERY_MAP`: check API key via `os.environ.get(_PROVIDER_KEY_MAP[provider], "")`, skip if empty with error message `f"Skipping {provider}: {key_name} not set"`
   - Use `ThreadPoolExecutor(max_workers=4)` + `as_completed`
   - Call `future.result(timeout=timeout)` to enforce outer timeout
   - Catch exceptions (TimeoutError, Exception) -> populate `errors[provider]`

9. **_get_fallback_models(provider: str) -> list[DiscoveredModel]:**
   - Import registry from `src.model_registry`
   - For each `model_id, mc` in `registry._models.items()` where `mc.provider == provider`
   - Return `DiscoveredModel(model_id=mc.model_id, provider=mc.provider, context_window=None, input_price_per_1m=mc.input_price_per_1m, output_price_per_1m=mc.output_price_per_1m)`

Use `logging.getLogger(__name__)` for all warnings. Type hints on all functions. Docstrings on all public functions.
