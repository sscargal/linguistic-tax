```python
def apply_intervention(
    prompt_text: str,
    intervention: str,
    model: str,
    call_fn: Callable[..., Any],
) -> tuple[str, dict[str, Any]]:
    match intervention:
        case "raw":
            return (prompt_text, {})
        case "self_correct":
            return (build_self_correct_prompt(prompt_text), {})
        case "prompt_repetition":
            return (repeat_prompt(prompt_text), {})
        case "pre_proc_sanitize":
            return sanitize(prompt_text, model, call_fn)
        case "pre_proc_sanitize_compress":
            return sanitize_and_compress(prompt_text, model, call_fn)
        case "compress_only":
            return sanitize_and_compress(prompt_text, model, call_fn)
        case _:
            raise ValueError(f"Unknown intervention: {intervention}")
```

    Create src/emphasis_converter.py with:

    1. **Code block protection helper** `_split_code_and_text(text: str) -> list[tuple[str, bool]]`:
       - Split text into segments: (content, is_code) tuples
       - Code blocks: triple-backtick fenced blocks, lines indented 4+ spaces
       - All conversion functions apply transforms only to non-code segments
       - Rejoin segments after transformation

    2. **Cluster A functions** (key-term emphasis -- take text + key_terms list):
       - `apply_bold_emphasis(text: str, key_terms: list[str]) -> str` -- wraps terms in `**term**`
       - `apply_caps_emphasis(text: str, key_terms: list[str]) -> str` -- converts terms to UPPER
       - `apply_quotes_emphasis(text: str, key_terms: list[str]) -> str` -- wraps in `'term'`
       - Sort key_terms by length descending before replacement to prevent partial matches
       - Use sentinel replacement to avoid double-matching (replace longest first with sentinel, then substitute)

    3. **Cluster B functions** (instruction-word emphasis):
       - `apply_instruction_caps(text: str) -> str` -- regex converts instruction verbs to ALL CAPS
       - Target words: "do not", "don't", "should", "must", "will", "shall", "need to", "have to", "return" (when followed by non-identifier chars)
       - Pattern for "return": `r'\breturn\b(?!\s*[(\[{a-z_A-Z])` to avoid matching "return value_name" code patterns
       - `apply_instruction_bold(text: str) -> str` -- regex wraps instruction verbs in **bold**
       - Same target words as caps, wrapped in `**word**`

    4. **Cluster C function**:
       - `lowercase_sentence_initial(text: str) -> str` -- lowercase first char after sentence boundary
       - Sentence boundaries: start of string, after `. `, `! `, `? ` (period/excl/question + whitespace)
       - Skip if next char is already lowercase or is part of an acronym (2+ consecutive uppercase)

    5. **Cache loader** for Cluster A pre-computed variants:
       - `load_emphasis_variant(prompt_id: str, intervention: str, cache_dir: str = "data/emphasis") -> str`
       - Reads from `{cache_dir}/cluster_a_{type}.json` where type is bold/caps/quotes
       - Returns the pre-converted prompt text for the given prompt_id
       - Raises FileNotFoundError if cache file missing (experiment hasn't been prepared)

    All functions: type hints, docstrings, use logging module (not print).
