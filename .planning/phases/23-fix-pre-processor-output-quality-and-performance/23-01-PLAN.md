<objective>
Implement noise-aware preprocessing skip logic, anti-reasoning system prompt hardening, and token-ratio warning logging.

Purpose: Fix the two root causes of preproc degradation -- (1) preprocessing clean/ESL prompts that models already handle natively, and (2) reasoning models inflating output tokens via hidden chain-of-thought. These changes eliminate the accuracy regression (73.6% -> 74.4%+ expected) and add observability for token bloat.

Output: Modified src/prompt_compressor.py (hardened prompts + token warning), modified src/run_experiment.py (skip logic), and comprehensive tests for both.
</objective>

2. Add skip logic BEFORE the match statement (after the docstring, before line 112). Insert:
```python
    # Skip preproc for non-type_a noise (clean and ESL handled natively by target models)
    if intervention in ("pre_proc_sanitize", "pre_proc_sanitize_compress", "compress_only"):
        if noise_type and not noise_type.startswith("type_a_"):
            return (prompt_text, {
                "preproc_skipped": True,
                "preproc_skip_reason": f"noise_type={noise_type}",
            })
```

Note: The condition `noise_type and not noise_type.startswith("type_a_")` ensures backward compatibility -- when noise_type is empty string (default, old callers), the skip does NOT trigger, so preproc runs as before.
